# DAIS Prospect Intelligence

Public-safe GitHub Pages briefs for Advancing Analytics DAIS prospect preparation.

## What this is

A lightweight, mobile-friendly briefing site for account conversations. Each page summarises:

- what the company does;
- why the named person matters;
- public company/data/AI context;
- Databricks relevance;
- Advancing Analytics conversation wedge;
- questions to ask;
- suggested next-best offers;
- source appendix.

## Safety rules

These pages are review drafts, not CRM records. They must exclude private contact data, commercial terms, commercial terms, private relationship notes, account-owner notes, private-message context and unsupported personal details.

Evidence labels used in pages:

- `verified` — public source supports the claim;
- `provided` — supplied by the account team;
- `inferred` — commercial hypothesis from evidence/context;
- `withheld` — intentionally not published.
